# Archana V Portfolio 🚀

A modern, futuristic, and tech-focused responsive personal portfolio web application built for an **Artificial Intelligence & Machine Learning Student**.

## ✨ Features

- **Futuristic AI Aesthetic**: Dark mode by default with neon blue and purple accents.
- **Glassmorphism Design**: Sleek cards and navigation with backdrop blur effects.
- **Typing Animation**: Dynamic hero section showcasing key skills (AI, ML, Cloud, DevOps).
- **Smooth Interaction**: Scroll-reveal animations for sections and project cards.
- **Project Showcase**: Professionally organized project cards with tech-stack tags.
- **GitHub Integration**: Dynamic stats and top languages cards via `github-readme-stats`.
- **Responsive Layout**: Optimized for both mobile and desktop viewing.
- **Dark/Light Mode**: Toggleable theme for accessibility.

## 🛠️ Tech Stack

- **Framework**: [Vite](https://vitejs.dev/) (Vanilla TypeScript)
- **Styling**: Vanilla CSS (Custom Variable-based Design System)
- **Icons**: Lucide
- **Analytics**: GitHub Stats API

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (Node Package Manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Archana487/portfolio-new.git
   cd portfolio-new
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
